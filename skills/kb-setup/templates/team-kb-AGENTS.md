Every AI agent operating on this team KB follows these rules.

## Identity

This is a **shared team knowledge base**. Multiple contributors write here. Each has their own directory (`<contributor>/inputs/`, `<contributor>/outputs/`). Shared decisions and TODOs live at the team root (`.decisions/`, `.tasks/`).

## Rules

1. **Never modify another contributor's directory** without explicit permission.
2. **RACIs are required** on team decisions and TODOs.
3. **Run the five-question gate** before persisting anything.
4. **Append inline changelog** on every `outputs/topics/` file update.
5. **Log every operation**.
6. **Suggest next steps**.
7. **Offer commit/push/PR** after substantive changes. Respect branch protection.
